COVision - COVID-19 Detection System

  1. Home

Project Information

  • Category: Deep Learning
  • Tools: CNN, Grad-CAM, Python, TensorFlow
  • Project Date: January 2024 - June 2024
  • GitHub: github.com/Divyansh1414
  • Dataset: Kaggle Dataset

Overview

This project applies deep learning techniques to detect COVID-19 positive cases from chest X-ray images. By categorizing images as COVID-positive, lung opacity, normal, or viral pneumonia, the system aids healthcare providers with quick and accurate diagnoses.

Key Highlights

  • Data Preprocessing: Grayscale conversion, augmentation, and Ben Graham's smoothing method.
  • Model: CNN trained with early stopping, dropout regularization, and Grad-CAM for interpretability.
  • Evaluation: Achieved high accuracy and reliable F1 scores with confusion matrix visualization.

Grad-CAM Visualization

Using Grad-CAM, the system provides heatmaps highlighting regions in X-ray images that influenced predictions, ensuring transparency and reliability for medical decisions.
