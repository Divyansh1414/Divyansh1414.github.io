TradeSync: Real-time Trade Processing Platform

TradeSync is a low-latency, cloud-native platform for processing and reporting financial trades in real time. I built the backend using Java and Spring Boot with Kafka-based streaming pipelines that improved throughput by 42% while maintaining strong ordering guarantees and exactly-once semantics for critical trade events.

For analytics and reporting I designed normalized schemas for PostgreSQL and Snowflake which reduced report lag by 35% and achieved 99.9% data accuracy. The frontend is a React + Redux dashboard containerized with Docker and deployed to Kubernetes via GitHub Actions CI/CD; the system delivers sustained latencies of <50ms for the critical trade paths and increased operational visibility by 3.2x.

Observability was implemented with Prometheus and Grafana, enabling actionable alerts and playbooks that significantly reduced mean time to recovery (MTTR). This project highlights my experience building real-time, resilient financial systems and end-to-end observability.
