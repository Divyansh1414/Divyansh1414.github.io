üìä Interactive Visual Analytics Dashboard

"Food Deserts and Health Inequality: A Visual Exploration of Nutrition Access in America" is a comprehensive visual analytics dashboard that explores how geography, poverty, and public health variables interact to shape food access and obesity rates across U.S. counties.

This project features interactive data visualizations with real-time linked brushing, advanced analytical methods (PCA, clustering, correlation analysis), and seamless storytelling in a scroll-free dashboard.

üìå Overview

This project was developed for CSE 564: Visual Analytics at Stony Brook University and meets the course requirements for interactive data visualizations, real-time linked brushing, advanced analytical methods, and seamless storytelling in a scroll-free dashboard.

üéØ Problem Statement

Millions of Americans live in "food deserts," with limited access to nutritious food. These areas often exhibit high obesity rates, child poverty, food insecurity, and chronic illness. Our goal was to use visual analytics to investigate how food access, income, and regional disparities drive health inequality in the U.S.

üìä Features

  • üåç Choropleth Map ‚Äì Adult Obesity
    County-level mapping with interactive pan, zoom, and click-to-select
  • üìâ PCA + KMeans Scatter Plot
    Reduces high-dimensional features to 2D and clusters counties into 3 health risk groups
  • üìå Custom Scatter Plot
    Plots any two user-selected variables with bubble size = Health Burden Index
  • ü™Ñ Parallel Coordinates
    Shows multivariate patterns across counties with interactive brushing
  • üßÆ Correlation Matrix
    Visualizes correlation between all variables (e.g., strong obesity-inactivity link at 0.80)
  • üç© Cluster Distribution Chart
    Donut chart shows population per cluster with clickable segments

üß† Health Burden Index

A composite metric used to summarize multiple health-related factors (e.g. obesity, smoking, food insecurity, income, inactivity) into a single value that indicates overall burden on health for each county. Used for:

  • Bubble size scaling
  • Analytical filtering
  • Targeting high-risk zones
