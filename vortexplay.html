VortexPlay: Scalable Video-On-Demand Platform

VortexPlay is a cloud-native video-on-demand streaming platform designed for scale. I built a single-page application with React and TypeScript for content tools enabling 800+ weekly beta sessions. On the backend, I integrated Golang services with Twirp, JWT authentication, and S3 uploads, improving ingest speed by 31% and reducing 500 errors by 42%.

For analytics, I deployed a Dockerized stack on AWS ECS with Terraform, using PostgreSQL, Redis, and Grafana dashboards to visualize real-time engagement, boosting mid-video completion rates by 22%. This project demonstrates my expertise in distributed systems, cloud infrastructure, and performance optimization.
